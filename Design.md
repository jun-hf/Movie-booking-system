# Movie booking system

## Specification
- Design a movie system that allows the users to view/book movie at <u>anywhere</u> and <u>anytime</u>.

## Use case
- Main actors
    1. Users
    2. Guest 
    3. Front officer
    4. Admin
    5. System

- Main use cases
    1. View movie at diffrent locations at diffrent cinema.
    2. Book / cancel / updates shows
    3. Book seats
    4. Make payments
    5. Cancel bookings
    6. View bookings
